Grid

A grid is a structure used to harmonize negative space in a layout

Definition
Standard Elements Types Variations

Standard

Grid

A grid is made up of columns of content with gutters to provide negative space between columns.

All grid systems must assume an arbitrary amount of column rows. Semantic by default assumes 16 columns

Elements

Rows

A row is used to create vertical padding between groups of columns on a page. If no rows are specified columns will sit flush vertically.

Columns

Types

Page Grid

A page grid exists on the top level of a web site. It can change its left and right gutter size to be responsive to a browser's width. This means the gutters to left and right of grid columns will alter in size as a browser's resolution increases.

Semantic UI's page grid by default will change the page gutters to increase at 250pixel intervals.
